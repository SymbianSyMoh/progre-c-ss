progre(c)ss

custom css progress bars with minimal effort!

progre(c)ss allows you to add progress bars/lines to your elements with minimal effort.

  • include the stylesheet
  • add the class progrecss to your element (any element)
  • add a data attribute data-progrecss defining the percent complete.
<div class="progrecss" data-progrecss="77">
 My first progre(c)ss!
</div>

That's it!

Helper classes.

There are some helper classes in place that will position your progre(c)ss bar where you want it.

  • bottom will place the progre(c)ss bar at the bottom.
  • top will place the progre(c)ss bar at the top(default).
  • fixed can be used in combination with those or alone to fix the position.
  • There are also some for some colors to get you started before hacking the source.
